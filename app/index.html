Hernia + estenosis + facetario coexisten. Preferencia extensión compite con flexión. El motor integra como lo haría un experto — pero en 30 segundos.

Chokepoints de seguridad
Catálogo entero verificado contra contraindicaciones. Si el paciente tiene osteoporosis severa, no verá plancha ni impacto. Automático, silencioso, no-negociable.

Evidence tiers honestos
Cada protocolo muestra S/A/B/C/D con cita. TENS: Tier A (Cochrane). Magnetoterapia: Tier D (sin evidencia). No vendemos pseudociencia — informamos.

Pain-Targeted Builder
Identifica patrones compensatorios (Trendelenburg + QL + kinesiofobia) y construye sesión analgésica dirigida en 7 fases. Multi-patrón simultáneo.

Síntesis diagnóstica progresiva
El diagnóstico no se escribe — se construye. Evaluación + imagen + comorbilidades + escalas → el motor sintetiza y sugiere. Tú confirmas.

Si el NPRS baja ≥2, el patrón sospechado se confirma para la próxima sesión. El sistema aprende de cada resultado. Ciclo que se auto-refuerza.
